Installing program

To install this program, you need to

git clone https://github.com/OrestasM/slic-segmentation.git

Running program

After cloning Github repo, you need to:\ Put your .SVS files in slic-segmentation folder, and after that, you can execute following commands

cd slic-segmentation
mkdir images
python deepzoom-multiserver.py

Now server should be up and running and you should access it by: 127.0.0.1:5000 Select your settings, press "Set" button and let the segmentation begin!
